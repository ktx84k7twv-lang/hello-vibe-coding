# 🎯 Vibe Coding - AI 오목

단일 페이지에서 AI와 오목을 즐길 수 있는 웹 서비스입니다.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

## 📋 프로젝트 소개

Vibe Coding의 첫 번째 프로젝트로, 간단하지만 완성도 있는 AI 오목 게임을 구현했습니다.
페이지 이동 없이 모든 플레이가 가능하며, 실제 서비스처럼 자연스러운 경험을 제공합니다.

## ✨ 주요 기능

### 1. AI 오목 대전
- 사용자(흑) vs AI(백) 1:1 대전
- 사용자가 항상 먼저 착수

### 2. 첫 수 제한 시간
- 게임 시작 후 **60초 이내**에 첫 수를 두어야 함
- 실시간 카운트다운 타이머
- 10초 이하일 때 경고 애니메이션

### 3. AI 난이도 선택
- **쉬움**: 1단계 탐색 깊이
- **보통**: 2단계 탐색 깊이 (기본값)
- **어려움**: 3단계 탐색 깊이

### 4. 자동 오목 룰
- 5개 연속 돌 배치 시 자동 승리 판정
- 가로, 세로, 대각선 모든 방향 체크
- 15x15 표준 오목판

## 🚀 시작하기

### 필요 사항
- 최신 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)

### 실행 방법

1. 저장소 클론
```bash
git clone <repository-url>
cd hello-vibe-coding
```

2. 브라우저로 실행
```bash
open index.html
# 또는 브라우저에서 index.html 파일을 직접 열기
```

3. 게임 시작
   - AI 난이도 선택
   - "게임 시작" 버튼 클릭
   - 60초 이내에 첫 번째 흑돌 착수
   - AI와 번갈아가며 플레이

## 🎮 게임 규칙

1. **흑돌(사용자)**이 먼저 시작합니다
2. 게임 시작 후 **1분 이내**에 첫 수를 두어야 합니다
3. 가로, 세로, 대각선으로 **5개의 돌**을 먼저 연속으로 놓으면 승리합니다
4. AI는 선택한 난이도에 따라 다른 전략을 사용합니다

## 🛠️ 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 모던 스타일링, Flexbox, Grid
- **JavaScript (ES6+)** - 게임 로직 및 AI 알고리즘
- **Canvas API** - 게임판 렌더링

## 🎯 AI 알고리즘

AI는 다음 전략을 순서대로 실행합니다:

1. **즉시 승리 가능한 수 탐색** - AI가 이번 턴에 이길 수 있다면 그 수를 둠
2. **상대방 승리 차단** - 사용자가 다음 턴에 이길 수 있다면 그 위치를 막음
3. **위치 평가** - 모든 가능한 수를 평가하여 최선의 수 선택
   - 연속된 돌의 개수
   - 양 끝이 열려있는지 여부
   - 중앙에 가까운 위치 가산점
